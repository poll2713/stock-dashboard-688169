## 任务2：技术指标说明

### RSI (相对强弱指数, Relative Strength Index)

**作用**：衡量价格上涨力量与下跌力量的相对大小，判断超买/超卖状态。

**计算方法** (Wilder 平滑法, 参数 N=14)：

```
1. 计算每日价格变动: ΔP = Close_today - Close_yesterday
2. 分离涨跌: 
   - Gain = max(ΔP, 0)    涨幅
   - Loss = max(-ΔP, 0)   跌幅
3. Wilder 平滑平均 (N=14):
   - AvgGain = (前日AvgGain × 13 + 今日Gain) / 14
   - AvgLoss = (前日AvgLoss × 13 + 今日Loss) / 14
4. 相对强弱: RS = AvgGain / AvgLoss
5. RSI = 100 - 100 / (1 + RS)
```

**判断标准**：
- RSI > 70：超买区，可能回调
- RSI < 30：超卖区，可能反弹
- RSI = 50：多空分界线

### MACD (指数平滑异同移动平均线)

**作用**：跟踪趋势方向和动量强度，通过快慢线交叉产生买卖信号。

**计算方法** (参数 12, 26, 9)：

```
1. 快线 EMA(12): EMA12 = Close × (2/13) + EMA12_前日 × (11/13)
2. 慢线 EMA(26): EMA26 = Close × (2/27) + EMA26_前日 × (26/27)
3. DIF (差离值): DIF = EMA12 - EMA26
4. DEA (信号线): DEA = DIF 的 9日 EMA
5. MACD 柱:     MACD = (DIF - DEA) × 2
```

**判断标准**：
- **金叉**：DIF 上穿 DEA → 买入信号
- **死叉**：DIF 下穿 DEA → 卖出信号
- MACD 柱由负转正：多头力量增强
- MACD 柱由正转负：空头力量增强

### 布林带 (Bollinger Bands)

**作用**：通过统计标准差构建价格通道，衡量波动率和相对价格水平。

**计算方法** (参数 20, 2)：

```
1. 中轨 (MB):  MA20 = 过去 20 日收盘价的算术平均
2. 标准差 (σ): σ = sqrt(Σ(Close_i - MA20)² / 20)
3. 上轨 (UP):  UP = MA20 + 2 × σ
4. 下轨 (DN):  DN = MA20 - 2 × σ
5. 带宽 (BW):  BW = (UP - DN) / MA20 × 100%
```

**判断标准**：
- 价格触及上轨：短期偏高，可能回调
- 价格触及下轨：短期偏低，可能反弹
- 带宽收窄：波动率降低，可能即将变盘
- 带宽扩大：波动率升高，趋势加速

### OBV (On Balance Volume, 能量潮指标)

**作用**：通过累计成交量来衡量买卖压力，验证价格趋势的可靠性。OBV 的核心理念是"量在价先"——成交量的变化往往先于价格变化。

**计算方法**：

```
初始 OBV = 0
对每个交易日 i (i > 0):
  如果 Close_i > Close_{i-1}:  OBV_i = OBV_{i-1} + Volume_i   (上涨日，加上成交量)
  如果 Close_i < Close_{i-1}:  OBV_i = OBV_{i-1} - Volume_i   (下跌日，减去成交量)
  如果 Close_i = Close_{i-1}:  OBV_i = OBV_{i-1}               (平盘日，不变)
```

**判断标准**：
- OBV 上升 + 价格上升 → 量价齐升，趋势健康
- OBV 下降 + 价格上升 → 量价背离，上涨可能乏力
- OBV 上升 + 价格下降 → 底部放量，可能即将反转
- OBV 下穿 OBV 均线 → 卖出信号
