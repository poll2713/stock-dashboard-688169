# 石头科技 (688169.SH) 行情数据看板

基于 Tushare 数据接口，对石头科技过去一年（2025-07-04 ~ 2026-07-03）的日线行情数据进行可视化分析的单页 HTML 看板。

## 在线预览

直接用浏览器打开 `index.html` 即可（需联网加载 ECharts CDN）。

## 项目结构

```
.
├── index.html                                    # 可视化看板（核心文件）
├── 688169_石头科技_daily_20250704_20260704.csv   # 原始日线数据
├── generate_dashboard.py                         # 看板生成脚本
└── README.md                                     # 本文件
```

## 看板内容

### 摘要卡片

展示最新收盘价、区间总收益率、区间最高/最低价、日均成交量、年化波动率、最大回撤、上涨/下跌天数统计、最大单日涨跌幅等关键指标。

### 图表

| 图表 | 说明 |
|------|------|
| **K线图 + 均线** | 日K线蜡烛图，叠加 MA5/MA10/MA20/MA60 四条均线 |
| **成交量** | 柱状图，红涨绿跌（A股惯例） |
| **MACD** | DIF 线、DEA 线 + MACD 柱状图 (参数 12,26,9) |
| **KDJ** | K 线、D 线、J 线 (参数 9,3,3) |
| **RSI** | RSI6、RSI12、RSI24 三条相对强弱指标线 |
| **布林带 BOLL** | 收盘价 + 上轨/中轨/下轨 (参数 20,2) |
| **日涨跌幅分布** | 直方图，按 1% 区间统计天数 |

## 技术指标说明

### MA (移动平均线)
N 日收盘价的算术平均值。MA5/MA10 反映短期趋势，MA20/MA60 反映中长期趋势。

### MACD (指数平滑异同移动平均线)
- DIF = EMA(12) - EMA(26)：快慢线之差
- DEA = EMA(DIF, 9)：DIF 的信号线
- MACD 柱 = (DIF - DEA) × 2：多空能量柱
- **金叉** (DIF 上穿 DEA) 视为买入信号，**死叉** (DIF 下穿 DEA) 视为卖出信号

### KDJ (随机指标)
- RSV = (收盘价 - 9日最低价) / (9日最高价 - 9日最低价) × 100
- K = 2/3 × 前日K + 1/3 × RSV
- D = 2/3 × 前日D + 1/3 × K
- J = 3K - 2D
- K > 80 超买区，K < 20 超卖区

### RSI (相对强弱指数)
衡量价格上涨力量与下跌力量的相对大小。RSI > 70 超买，RSI < 30 超卖。

### BOLL (布林带)
- 中轨 = MA20
- 上轨 = MA20 + 2σ
- 下轨 = MA20 - 2σ
- 价格触及上轨可能回调，触及下轨可能反弹

## 技术栈

- **图表库**: [Apache ECharts 5.5](https://echarts.apache.org/)
- **数据处理**: Python 3.13 + 标准库 (csv, json)
- **数据源**: Tushare Pro API
- **前端**: 原生 HTML + CSS + JavaScript（无框架依赖）

## 颜色惯例

遵循 A 股市场惯例：
- **红色** = 上涨 / 收阳
- **绿色** = 下跌 / 收阴

## 使用方法

### 重新生成看板

如需修改指标参数或更新数据，编辑 `generate_dashboard.py` 后重新运行：

```bash
python generate_dashboard.py
```

脚本会读取同目录下的 CSV 文件并重新生成 `index.html`。

### 更换股票

1. 通过 Tushare 获取目标股票的日线数据，保存为 CSV
2. 修改 `generate_dashboard.py` 中的 `CSV_PATH` 和标题
3. 重新运行脚本
